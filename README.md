# NSS VIT - Official Website

> **"Not Me, But You"** - The motto that drives our community service

The official repository for the National Service Scheme (NSS) unit at Vidyalankar Institute of Technology, Mumbai. This website showcases our activities, events, achievements, and the impact we're making in the community.

## Features

- **Homepage** - Hero section, highlights, stats, about NSS, FAQs
- **Events** - Activities organized by category and academic year
- **Team** - Leadership and volunteers with year-wise filtering
- **Gallery** - Photo collections with lightbox viewing
- **Camp** - Special Camp information and impact stories
- **Achievements** - Awards and recognitions in competitions
- **Get Involved** - Contact form and ways to contribute
- **Admin Panel** - Content management via Sanity Studio

## Tech Stack

### v2.0 (2025-2026) - Current
- **Framework:** [Astro](https://astro.build) v5
- **UI Components:** [React](https://react.dev) v19
- **Styling:** [Tailwind CSS](https://tailwindcss.com) v4
- **CMS:** [Sanity](https://sanity.io) v5
- **Animations:** [Framer Motion](https://framer.com/motion)
- **Deployment:** [Vercel](https://vercel.com)

### v1.0 (2024) - Legacy
- React.js + Vite + TailwindCSS

## Development

```bash
# Navigate to the project
cd nss-vit

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
